# jciai

A deployed AngularJS website for JCI Amsterdam International, built in January 2015 as a volunteer contribution to the chapter.

## Stack

- AngularJS (Yeoman-generated scaffold)
- Gulp build pipeline
- SASS
- Karma for unit tests, Protractor for end-to-end tests

## Structure

- `src/app/main/`: landing page and main controller
- `src/app/content/`: board pages, JCI content, old-boards archive
- `src/components/navbar/`, `src/components/footer/`: shared UI components
- `e2e/`: Protractor specifications
- `src/app/main/main.controller.spec.js`: unit-test example

## Context

Built during my 2012-2015 volunteer membership with JCI Amsterdam International (focus: Economic Empowerment). Pro bono project; shipped and used by the chapter for event coordination and board communication. Not maintained since 2015.
